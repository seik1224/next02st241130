# Next js
 
- React를 기반으로 한 프레임워크로 SSR, CSR 모두 가능
- 풀스택 프레임워크로 백엔드 개발 가능
- 클라이언트 측에서만 실행되어야 하는 코드 명시
'use clinet'
- src/app안에 tailwindCSS가 적용되는 파일들을 넣는다
- 14.2.4 버전으로 다운
